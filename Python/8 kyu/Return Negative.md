## Return Negative


In this simple assignment you are given a number and have to make it negative. But maybe the number is already negative?




---

### Given Code


```python
def make_negative( number ):
  pass
   
```

---

### Solution


```python
def make_negative( number ):
    if number  >= 0:
        return number * -1
    elif number < 0:
        return number
    
```
