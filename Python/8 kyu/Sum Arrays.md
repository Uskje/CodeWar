## Sum Arrays



Write a function that takes an array of numbers and returns the sum of the numbers. The numbers can be negative or non-integer. If the array does not contain any numbers then you should return 0.

---

### Given Code

```python
def sum_array(a):
  pass
```
---

### Solution

```python
def sum_array(a):
  return sum(a)
    
```
