## Opposite number


**Description:**

Very simple, given a number, find its opposite.


**Examples**

```
1: -1
14: -14
-34: 34
```



---

### Given Code


```python
def opposite(number):
    pass
```

---

### Solution


```python
def opposite(number):
    return number * -1
```
